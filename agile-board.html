Agile Task Management Application

TThe Task Manager Application is a web-based tool designed to streamline task management and enhance productivity. It offers users a comprehensive platform for organizing tasks, managing projects, and optimizing workflow efficiency.

Additionally, the application provides dynamic theme switching, empowering users to tailor their experience with options for both light and dark themes, ensuring optimal readability in any environment. With its intuitive user interface, navigating the app and interacting with tasks is a breeze, enhancing user experience. Integration with local storage enables the persistence of user data across sessions, ensuring convenience and accessibility by allowing users to pick up where they left off effortlessly.

Technologies Used: HTML, CSS (including Tailwind) and JavaScript
